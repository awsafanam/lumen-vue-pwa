# Lumen Vue PWA

Important: Run the install command as administrator

## Project setup
- Lumen 5.7
- Vue Cli 3
- PWA
- Hot Reload
- Vuex
- Vue Router
- Vuetify
- Layouts
- vForm
- User Authentication


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Start production Server
```
npm start
```
